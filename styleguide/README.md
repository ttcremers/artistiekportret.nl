# Artistiekportret.nl visual contract

The files in this directory are the design reference for the production website.

## Live design workspace

- Root design workspace: http://zone.photos/artistiekportret.nl/
- Homepage prototype: http://zone.photos/artistiekportret.nl/styleguide/
- Style guide: http://zone.photos/artistiekportret.nl/styleguide/style-guide.html

## Purpose

Use the prototype as the visual source of truth before changing production templates.

Use the style guide as the implementation contract for:

- colors
- typography
- interactive elements
- image treatment
- responsive behavior
- explicit do/don't rules

## Workflow

Before implementing or redesigning a production page:

1. Read the style guide.
2. Inspect the homepage prototype at desktop, tablet and mobile widths.
3. Preserve the established visual grammar unless a deliberate art-direction change has first been made in the prototype.
4. Keep production work separate from visual experimentation.

## Responsive hero rules

- Desktop and tablet: title, portrait and note remain separate columns.
- Title and portrait may sit close together but must never overlap.
- The hero composition is top-anchored below the eyebrow; extra vertical whitespace belongs below the work, not above it.
- The hero note stays readable without decorative linework behind it.
- Mobile: title, portrait and note stack cleanly.
